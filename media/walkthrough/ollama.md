# Install Ollama

AskFirst uses a local AI model through [Ollama](https://ollama.com), so refining prompts is free and nothing leaves your machine.

1. Download Ollama from **ollama.com** (macOS, Windows, Linux)
2. Open it once — it runs quietly in the background

Prefer a cloud model instead? Run **AskFirst: Set API Key** from the Command Palette and switch `askfirst.provider` in Settings.
