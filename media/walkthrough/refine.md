# Refine your first prompt

Three ways to start:

- Select rough prompt text in any file, right-click → **AskFirst: Refine Prompt**
- Press `Cmd+Alt+R` (`Ctrl+Alt+R` on Windows/Linux)
- Click **✨ AskFirst** in the status bar

Answer the 3–5 questions (pick an option, type your own, or skip — use the ← button to change an earlier answer). The refined prompt opens beside your editor, ready to paste into any AI.
